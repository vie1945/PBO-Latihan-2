# PBO-Latihan-2
Praktikum Pemrograman Berorientasi Objek - Java
Repository ini berisi hasil latihan praktikum mengenai Object-Oriented Programming (OOP) menggunakan Java.

Deskripsi
Pada praktikum ini, dilakukan implementasi konsep Class & Object melalui pembuatan sistem kontrol televisi sederhana. Program ini mendemonstrasikan bagaimana atribut (state) dan metode (behavior) bekerja dalam sebuah kelas objek, lengkap dengan validasi logikanya.

Struktur Kode
TV.java: Berisi definisi kelas TV, atribut (channel, volume, status on/off), dan metode untuk mengontrol TV.

TestTV.java: Kelas utama (Main Class) untuk melakukan instansiasi objek dan menjalankan simulasi program.

Tugas Praktikum
Repository GitHub: Mengunggah semua kode latihan sesuai materi perkuliahan.
